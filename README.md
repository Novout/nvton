## Features

- ✅ Vue and Nuxt Plugin
- ✅ Local or File Object
- ✅ Support JSON
- ✅ Exclude Wrong Data
- ✅ Supports Node v20+

## Syntax

```ts
[
  'foo',
  true,
  undefined,
  0,
  nan,
  infinity,
  -infinity,
  { foo: 0 },
  [
    ['key1' | 'value'],
    ['key2' | 0],
    ['key3' | false], 
    [4 | null],
    ['key5' | { bar: 0 }]
  ]
]
```
